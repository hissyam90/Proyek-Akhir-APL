Repositori ini merupakan tempat pengumpulan **Program**, **Flowchart**, dan **Laporan** untuk **Proyek Akhir** mata kuliah **Algoritma dan Pemrograman Lanjut (APL)** oleh **Kelompok 6**.

## 📂 Deskripsi Folder

- **Program/** Berisi file kode sumber (`.cpp`) dan header (`.h`) dari sistem manajemen pegawai yang dikembangkan menggunakan bahasa C++. Sistem ini mencakup fitur autentikasi, manajemen data (CRUD), dan sistem presensi.

- **Flowchart/** Berisi diagram alur (`.drawio`) yang menggambarkan logika dan proses kerja sistem, mulai dari proses login hingga fungsi-fungsi khusus admin dan user.

- **csv/** Berisi penyimpanan data dari program ini dalam format CSV, mencakup data akun, profil pegawai, dan riwayat absensi untuk memastikan persistensi data.
